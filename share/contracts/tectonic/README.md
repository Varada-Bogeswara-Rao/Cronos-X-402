# Tectonic Verified Contracts

This directory contains the exact source code of the Tectonic Finance protocol, verified on Cronoscan.
We use this for high-fidelity simulation ("Heavy Mock").

## Files Needed
1. `TectonicCore.sol` (Comptroller)
2. `TectonicOracle.sol` (Price Oracle)
3. `InterestRateModel.sol` (WhitePaperInterestRateModel)
4. `tUSDC.sol` (CErc20Delegate + Delegator)
